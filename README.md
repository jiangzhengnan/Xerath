### 1.简介
AOP为Aspect Oriented Programming的缩写，意为：面向切面编程。
本项目为AOP思想在Android上的运用与实战，参考基于AspectJ的开源项目[AopArms](https://github.com/AICareless/AopArms)
旨在通过注解+Transfrom+ASM实现一套功能强大，方便开发，并且能够有效减少重复代码的Android Aop 框架。
